3) Short Report (one page)

The Campus Info Portal is a compact web application built for college use. It provides students with bus timings, canteen menu, library info, notices and a simple attendance interface. The application is intended as a beginner project demonstrating client-side data storage (localStorage) and simple UI routing using JavaScript. Future improvements include server-side storage, authentication, and mobile responsiveness.
